Supervisor có thể 
-> Approve (with review or not) 
	-> Nếu có deadline thì set status của submission = approved 
		-> set status của task = in progress
	-> Nếu không deadline thì set status của submission = approved + status của task = done

- Rejected 
-> bắt buộc có review và deadline mới
-> set status của submission = rejected
-> chuyển lại về trang task management
						

Staff có thể:
- Add submission -> chỉ khi chưa có submission nào 
- Remove submisson -> set status task = in progress
